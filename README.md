# Petersplain

Creates TikTok videos of Peter Griffin explaining anything, using MoviePy and
ElevenLabs.

## Installation

```bash
# Clone the repo
git clone git@github.com:hmzdot/petersplain.git
cd petersplain

# Install dependencies with uv
uv sync

# Run the program
uv run src/main.py {TEXT} -o {OUT_FILE}
```
